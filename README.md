# Recycling Management System

A comprehensive web application for managing recycling operations, including waste collection, inventory tracking, and reporting.

## Features

- **Transaction Management**: Record weighing, costing, and payment details
- **Inventory Tracking**: Real-time monitoring of material inventory
- **Multi-location Support**: Manage multiple collection locations
- **Waste Picker Management**: Track waste pickers and their transactions
- **Apartment Complex Integration**: Connect with residential complexes
- **Offline Support**: Work without internet connection with automatic sync
- **Comprehensive Reporting**: Generate detailed reports with filters
- **User Management**: Role-based access control
